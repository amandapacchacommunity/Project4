Advancement Portfolio and Donor Retention Analysis

A synthetic analytics project focused on membership participation, pledge fulfillment, donor engagement patterns, and core fundraising performance within a nonprofit advancement context.

View GitHub Repository

Project Overview

This project demonstrates how donor and fundraising data can be organized, validated, and explored to support advancement decision-making.

The analysis uses synthetic datasets to model donor records, membership activity, pledge status, and high-level fundraising indicators across a nonprofit advancement workflow.

What the Notebook Covers

Data Preparation

Loads analysis libraries and creates reproducible synthetic datasets for donor, membership, and pledge analysis.

Validation

Checks dataset size, structure, and sample records to support clean downstream analysis.

Initial Insights

Visualizes membership level distribution, pledge status, and fundraising indicators to highlight early patterns.

Example Analysis Questions

  • How are membership levels distributed across the donor base?
  • What share of pledges are paid versus outstanding?
  • How much modeled fundraising value is tied up in outstanding pledges?
  • How do campaign revenue patterns shift across years?
Current scope:

This version focuses on setup, validation, dashboard KPIs, and introductory visuals. Future pages can add donor segmentation, retention rates, and portfolio coverage analysis.

Tools Used
