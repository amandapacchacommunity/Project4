Advancement Portfolio and Donor Retention Analysis

A synthetic analytics project focused on membership participation, pledge fulfillment, and donor engagement patterns within a nonprofit advancement context.

View GitHub Repository

Project Overview

This project demonstrates how donor and fundraising data can be organized, validated, and explored to support advancement decision-making.

The analysis uses synthetic datasets to model donor records, membership activity, and pledge status across a nonprofit fundraising workflow.

Synthetic Data Nonprofit Analytics Membership Trends Pledge Tracking

What the Notebook Covers

Data Preparation

Loads analysis libraries and creates reproducible synthetic datasets for donor, membership, and pledge analysis.

Validation

Checks dataset size, structure, and sample records to support clean downstream analysis.

Initial Insights

Visualizes membership level distribution and pledge status to highlight early fundraising patterns.

Example Analysis Questions

  • How are membership levels distributed across the donor base?
  • What share of pledges are paid versus outstanding?
  • How can synthetic advancement data support repeatable analysis workflows?
Current scope:

This version focuses on setup, validation, and introductory visuals. Future pages or notebooks can add retention rates, donor segmentation, and portfolio coverage analysis.

Tools Used

Python, Pandas, NumPy, Matplotlib, Seaborn, Google Colab, and GitHub.
